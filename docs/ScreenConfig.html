Class: ScreenConfig

ScreenConfig

extends ClassConfigBase

This is the class to configure an object which simulates Screen in HTML DOM.

An instance of this class represents a configuration of a monitor device on which a Web browser is displayed.

new ScreenConfig(initConfig)

Creates a new instance of this class.
